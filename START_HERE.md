## 📁 KEY FILES TO KNOW

### Easy to Edit (No Coding!)
```
/config/images.js                    ← Images & social media
/config/departmentQuestions.js       ← Interview questions
```

### Upload Your Images Here
```
/public/images/carousel/             ← Gallery photos
/public/images/departments/          ← Department photos
/public/images/social/               ← Social icons
```

### Don't Edit (Unless You Know Code)
```
/app/page.tsx                        ← Landing page code
/app/verify/page.tsx                 ← Verification code
/app/apply/[dept]/page.tsx           ← Application form code
/app/globals.css                     ← Styling
```
